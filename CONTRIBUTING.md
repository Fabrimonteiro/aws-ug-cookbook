# Contributing to AWS User Group Cookbook

Thank you for considering contributing to the AWS User Group Cookbook! We welcome contributions that help improve this resource for the global AWS community.

## How Can You Contribute?

- **Share Best Practices**: Add your own strategies for organizing successful events.
- **Submit Case Studies**: Share your user group's success stories or lessons learned.
- **Provide Templates**: Contribute templates that others can use.
- **Enhance Documentation**: Improve existing content for clarity and completeness.

## Steps to Contribute

1. **Fork the Repository**: Click the 'Fork' button at the top right of the page.
2. **Clone Your Fork**: `git clone https://github.com/global-aws-community/aws-ug-cookbook.git`
3. **Create a New Branch**: `git checkout -b feature/YourFeature`
4. **Make Your Changes**: Add or edit content.
5. **Commit Your Changes**: `git commit -m 'Add feature'`
6. **Push to Your Fork**: `git push origin feature/YourFeature`
7. **Submit a Pull Request**: Go to your fork on GitHub and click 'New pull request'.
